# TodoContextLocal

A lightweight task management application built with React, Context API, and Local Storage. The project demonstrates scalable state management patterns and persistent client-side data storage without relying on external state management libraries.

## Features

- Create tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Persistent storage using Local Storage
- Global state management with Context API
- Responsive UI

## Tech Stack

- React
- Context API
- JavaScript (ES6+)
- CSS
- Local Storage

## Project Structure

src/
├── components/
├── context/
├── hooks/
├── pages/
├── utils/
└── App.jsx

## Installation

```bash
git clone https://github.com/yourusername/todo-context-local.git

cd todo-context-local

npm install

npm run dev
```

## How It Works

The application uses React Context API to manage global task state. Task data is synchronized with Local Storage, allowing users to retain their tasks even after refreshing the browser.
